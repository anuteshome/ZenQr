Restaurant QR Code Ordering & Management System
A responsive web application designed for restaurants, with three main sides:
Customer Side

Each table has a unique QR code linked to that specific table number
Customer scans the QR code → automatically taken to the digital menu with the table number already identified
They browse the menu and place their order directly from their phone
After ordering, they can track their order status in real-time (e.g. Placed → Preparing → Ready → Served)
Bilingual support — customers can switch between English and Amharic on the menu
No need to call a waiter at all

Kitchen Display Screen

A dedicated screen just for the kitchen
Shows all incoming orders in real-time with the table number
Kitchen staff can update the order status (Preparing → Ready)
Helps the kitchen stay organized and prioritize orders

Admin Side

Real-time Notifications — instant alert when a new order comes in, showing table number and items ordered
Manage Menus — add, edit, remove food items, categories, prices, photos, and menu item names in both English and Amharic
Manage Employees — track staff, roles, shifts, etc.
Manage Inventory — track ingredients and stock levels
Admin panel also available in both English and Amharic
